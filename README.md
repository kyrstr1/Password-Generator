## 📌 About

This is a simple browser-based password generator.

Users select the character types they want, then click one of the fixed-length buttons (8, 12, or 16).  
After clicking a button, a password is instantly generated.

When you click on the generated password, it is automatically copied to your clipboard.

No backend. No data storage. Everything runs in the browser.

---

## ⚙️ How It Works

1. Select the character types you want to include:
   - Uppercase letters
   - Lowercase letters
   - Numbers
   - Symbols

2. Click one of the length buttons:
   - **8**
   - **12**
   - **16**

3. A random password is generated.

4. Click on the generated password → it copies automatically.
